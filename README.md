# TIC-TAC-TOE-AI
Implemented an AI bot for playing TIC-TAC-TOE using the mini-max algorithm.

## Steps to run
1. Clone the repo
2. cd into the root folder
3. python3 main.py
4. Give desired role for starting the game HUMAN/AI.
